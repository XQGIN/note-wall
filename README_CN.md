# Note Wall - 温馨提醒墙

[English Documentation](README.md) | [中文文档](README_CN.md)

一个美观、交互式的网页应用，以视觉震撼的玻璃拟态界面展示积极向上的鼓励信息。

## ✨ 功能特性

- **玻璃拟态设计**: 现代化的玻璃效果窗口，带有流畅的动画效果
- **动态背景**: 渐变背景，持续色彩变换
- **响应式设计**: 完美适配桌面、平板和移动设备
- **自动主题切换**: 根据时间自动切换白天和夜间主题
- **交互元素**: 鼠标跟随光效和浮动动画
- **积极信息**: 50+ 条中文鼓励和励志信息
- **触摸友好**: 针对触摸设备优化尺寸

## 🚀 快速开始

### 方法1：直接文件访问
```bash
# 克隆仓库
git clone https://github.com/XQGIN/note-wall.git

# 打开主文件
直接双击打开 note-wall/index.html
```

### 方法2：本地服务器（推荐）
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js（如果安装了 http-server）
npx http-server -p 8000

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000/note-wall/`。

## 🎮 使用方法

1. **开始体验**: 点击"你准备好了吗？"按钮
2. **观看效果**: 150+ 个玻璃窗口将显示积极信息
3. **交互功能**:
   - **左键点击**: 开始/重新开始提醒墙
   - **右键点击**: 切换白天/夜间主题
   - **鼠标移动**: 玻璃窗口会响应鼠标位置
   - **点击窗口**: 关闭单个信息窗口

## 📱 响应式设计

应用自动适配不同屏幕尺寸：

- **桌面** (>768px): 完整的玻璃拟态体验
- **平板** (480px-768px): 针对触摸交互优化
- **手机** (320px-480px): 小屏幕的紧凑布局
- **超小屏幕** (<320px): 极小屏幕的简约设计

## 🎨 主题系统

### 自动主题切换
- **白天主题** (6:00-18:00): 明亮、多彩的渐变
- **夜间主题** (18:00-6:00): 暗色、平静的渐变

### 手动控制
- 右键点击主按钮切换主题
- 主题偏好保存在 localStorage 中

## 🛠️ 技术细节

- **纯 HTML/CSS/JavaScript**: 无外部依赖
- **CSS 变量**: 动态主题系统
- **CSS Grid & Flexbox**: 现代化布局技术
- **CSS 动画**: 流畅的过渡和效果
- **本地存储**: 主题偏好持久化
